# novigngn

Telegram-бот для проверки баланса API-ключей gngn.my.

## Запуск

```bash
pip install -r requirements.txt
python bot.py
```

Токен можно переопределить через переменную окружения `TELEGRAM_TOKEN`.

## Использование

Отправь боту API-ключ (начинается с `sk-`) — в ответ придёт баланс, статус, план и разбивка по моделям.
